# Ebook & Long-Form Writing Guide

## Structure

- Start with a compelling hook that promises a benefit or solves a problem.
- Use the "inverted pyramid" for blog posts: most important info first, then supporting details, then background.
- For ebooks: Introduction → 3-5 core chapters → Conclusion + Call to action.
- Each chapter should end with a summary and a teaser for the next.

## Tone & Voice

- Conversational yet professional. Use contractions ("you'll", "it's") to sound natural.
- Ask rhetorical questions to engage the reader.
- Prefer short paragraphs (2-4 sentences) and active voice.
- Use the "you" perspective to speak directly to the reader.

## Pacing & Flow

- Vary sentence length to create rhythm.
- Use subheadings to break up text and guide the skimming reader.
- Add concrete examples, analogies, and stories to illustrate abstract points.
